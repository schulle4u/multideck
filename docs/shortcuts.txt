MultiDeck Audio Player - Keyboard Shortcuts
===========================================

FILE MENU
---------
Ctrl+O              Load project
Ctrl+S              Save project
Ctrl+Shift+S              Save project as
Alt+F4              Exit application

MODE SELECTION
--------------
F3              Mixer Mode
F4              Solo Mode
F5              Automatic Mode

DECK CONTROLS
-------------
F6                  Jump to deck selection list
Ctrl+1 to Ctrl+0    Jump to deck 1-10
Ctrl+Tab            Next deck (in Solo/Automatic mode)
Ctrl+Shift+Tab      Previous deck
Ctrl+M              Mute/Unmute active deck
Ctrl+L              Toggle loop for active deck
Ctrl+F              Load file into active deck
Ctrl+Shift+F        Load URL into active deck
F2                  Rename active deck
Delete              Unload active deck

VOLUME & BALANCE
----------------
Ctrl+Up/Down        Increase/decrease deck volume
Ctrl+Left/Right     Adjust deck balance left/right
Ctrl+Shift+Up/Down  Increase/decrease master volume

DECK CONTEXT MENU
-----------------
Applications Key or Shift+F10    Open context menu for active deck

RECORDER
--------
Ctrl+R              Start/Stop recording

VIEW
----
Ctrl+T              Toggle status bar
Ctrl+Shift+T        Toggle theme (Light/Dark)

TOOLS
-----
Ctrl+P              Open options dialog

HELP
----
F1                  Show this keyboard shortcuts reference

NAVIGATION
----------
Tab                 Move to next control
Shift+Tab           Move to previous control
Space/Enter         Activate button/control
Escape              Close dialog/cancel

For more information, visit the project documentation.
